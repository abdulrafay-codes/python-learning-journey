### 1. Port Scanner Audit Tool (port_scanner.py)
A Python-based terminal script designed to audit network ports against a list of known secure configurations. It processes user inputs, handles input exceptions, and flags potential security vectors.

* *Key Features:* Loop auditing, standard port validation, and automated exception logging.
* *How to Run:*
```bash
  python port_scanner.py
`` `
